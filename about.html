About

WhatsTheDigit is a handwritten digit recognition tool built using neural networks. It is written in Python and uses the pyneural library. It is trained on the 60,000 images from the MNIST dataset. The front-end web app and the implementation of the neural network were both written by Kevin Holland as part of a research project at Capital University in Bexley, OH.
